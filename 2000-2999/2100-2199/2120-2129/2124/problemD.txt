Then, delete $$$a_i$$$ from $$$a$$$, concatenating the remaining parts of the array. Determine if it is possible to get an array that is a palindrome$$$^{\text{∗}}$$$ after any number of operations. Note that an empty array is considered a palindrome. $$$^{\text{∗}}$$$An array $$$b=[b_1,b_2,\ldots,b_m]$$$ is a palindrome if for each $$$1 \leq i \leq m$$$, $$$b_i=b_{m+1-i}$$$.InputEach test contains multiple test cases.
